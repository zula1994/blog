CSS3 clip-path

CSS 属性使用裁剪方式创建元素的可显示区域。区域内的部分显示，区域外的隐藏。

1 . 属性介绍

clip-path 属性可以创建一个只有元素的部分区域可以显示的剪切区域。区域内的部分显示，区域外的隐藏。

剪切区域是被引用内嵌的URL定义的路径或者外部svg的路径，或者作为一个形状例如circle()。

2. 属性使用

  1. 圆形 clip-path: circle(50% at 50% 50%); //circle(半径 at 圆心坐标)

    .path4 {
    width: 100px;
    height: 100px;
    position: relative;
    background: #2e7f40;

    content: "mask";
    color: #2e7f40;
    position: absolute;
    width: inherit;
    height: inherit;

}
